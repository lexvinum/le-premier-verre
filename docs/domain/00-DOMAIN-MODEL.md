# Le Premier Verre — Domain Model

Le Domain Model définit les objets fondamentaux de la plateforme.

Chaque nouvelle fonctionnalité devra utiliser ces objets plutôt que d'en créer de nouveaux.

---

# Entités principales

## 🍾 Wine

Représente une bouteille précise.

Exemples :

- Clos Jordanne Chardonnay 2022
- Le Grand Clos Pinot Noir 2021
